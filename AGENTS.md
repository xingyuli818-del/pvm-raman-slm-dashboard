# Codex Project Rules

## 项目背景

这是一个用于科研汇报的本地 HTML 交互式展示项目，服务于原位晶体转晶在线监测、PVM 图像识别、YOLO 实例分割、Raman 终点分析、SLM 标定展示和组会汇报。

## 技术规则

- 优先使用原生 HTML、CSS、JavaScript。
- 不要默认引入 React/Vue/Vite，除非用户明确要求。
- 所有路径使用相对路径。
- 页面必须能通过 `python -m http.server 8000` 本地预览。
- 不要写死 Windows 绝对路径。
- 不要删除已有数据文件。
- 修改前先检查当前目录结构。
- 修改后说明改了哪些文件、如何运行、如何验证。

## 页面风格

- 16:9 汇报展示友好。
- 中文标签。
- 科研组会风格。
- 简洁、清晰、少装饰。
- 图表适合截图放入 PPT。

## 数据规则

- CSV 默认放在 `./data/`。
- 图像默认放在 `./assets/` 或 `./figures/`。
- 如果 CSV 列名不匹配，先写兼容逻辑或提示需要的列名。
- 不要假设真实数据一定存在，必须提供 example CSV。

## 分析规则

- PVM 终点判定优先使用 `block_frequency`、滑动窗口、阈值、持续性和无回弹逻辑。
- Raman 终点判定优先使用 `1624 cm^-1` 强度、平台期、相对波动、斜率和持续性逻辑。
- 图像终点和 Raman 终点需要在同一时间轴上对齐展示。
